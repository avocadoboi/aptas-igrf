The $n$th degree, $m$th order associated legendre function is defined as
\begin{equation}
  P_n^m(\mu) = \alpha_n^mD^{m+n}(\mu^2-1)^n
\end{equation}
where
\begin{equation}
  \alpha_n^m = \sqrt{(2-\delta_{m0})\frac{(n-m)!}{(n+m)!}}\frac{1}{2^nn!}(1-\mu^2)^{m/2}
\end{equation}
